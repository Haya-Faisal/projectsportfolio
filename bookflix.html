Features

Interactive 3D Visualization System

  • p5.js 3D Engine: Dynamic floating box animations with real-time user interaction.
  • Object Pooling System: Manages up to 600 animated boxes with efficient lifecycle management.
  • Spatial Partitioning Grid: Optimizes neighbor detection and collision performance (60px grid cells).
  • Physics Simulation: Wave propagation, noise-based movement, and neighbor influence algorithms.

Book Search & Discovery Platform

  • Google Books API Integration: Full-volume search capabilities with rich metadata.
  • Real-time Typeahead Suggestions: Showing covers, authors, and ratings as you type.
  • Multi-page Navigation Flow: Search → details → AI analysis → recommendations.
  • Genre-based Discovery: Curated Goodreads genre links for exploration.

AI-Powered Content Analysis

  • Ollama LLM Integration: Book summaries using multiple models (Llama 2, Qwen).
  • Intelligent Recommendations: Based on user preferences and book characteristics.
  • Dynamic Content Generation: Formatted output with bold titles and author names.

// Ollama integration for AI features
async function generateBookSummary(req, res) {
    const { bookTitle } = req.body;
    const response = await ollama.generate({
        model: 'llama2',
        prompt: `Write a detailed summary of the book "${bookTitle}"...`
    });
    res.json(response);
}

Challenges & Solutions

Challenge: Real-time Search Suggestions

Solution: Used debouncing (300ms) and efficient DOM updates to show suggestions without overwhelming the API or UI.

Challenge: AI Response Formatting

Solution: Created regex-based formatting functions to consistently bold book titles and authors in AI-generated content.

Challenge: Large AI Models

Solution: Use qwen2.5:0.5b for faster replies with a fallback to larger models when quality is prioritized.
